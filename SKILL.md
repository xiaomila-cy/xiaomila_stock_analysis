# 金叉 · A股分析框架 v1.0

> 本文档定义金叉在股票/行业分析时的完整工作流。
> 目标：给一个股票或行业，知道拉什么数据、做什么分析、输出什么结构。

## 一、触发场景与入口判断

用户输入类型 | 触发模式 | 说明
---|---|---
"分析下XX股票" | **个股深度分析** | 单只股票全维度拆解
"XX行业怎么看" | **行业全景分析** | 行业宏观+板块横评+标的筛选
"这三只哪个好" | **多股对比分析** | 横向PK + 排名输出
"最近XX跌了好多" | **事件驱动分析** | 先查跌因，再判断是错杀还是基本面恶化

## 二、数据采集层 —— 拉什么数据

### 2.1 行情数据（腾讯API）`qt.gtimg.cn`

```
必拉字段：
  [3] 现价      [4] 昨收       [5] 今开
  [6] 成交量(手) [32] 涨跌幅%   [33] 最高  [34] 最低
  [38] 换手率    [39] PE(TTM)   [43] 振幅%  [44] 总市值(亿)

周期涨跌幅（GP-A定位法）：
  GP-A +8  = 近1月涨跌幅
  GP-A +9  = 近3月涨跌幅
  GP-A +10 = 近6月涨跌幅
```

单次查多个股票：逗号分隔 `qt.gtimg.cn/q=sz002460,sz002466,sz002240`

### 2.2 行业/商品现货价格

数据 | 来源 | 方法
---|---|---
碳酸锂/铜/稀土现货 | 东方财富搜索 `so.eastmoney.com` | browser_navigate + 提取Mysteel/生意社报价
期货价格 | 同上 | 搜索"XX期货主力"
行业库存/产量 | 同上 | 富宝资讯、Mysteel等第三方数据

### 2.3 政策消息验证

步骤 | 操作
---|---
1 | 拆解用户传播的消息为多条可验证陈述
2 | `browser_navigate("https://so.eastmoney.com/news/s?keyword=关键词")`
3 | 逐条对照原文，用 ✅/❌ 标记真伪
4 | 区分"硬性规定"vs"鼓励方向"

不可用：百度（滑块验证）、搜狗（点击验证）、雪球API（需登录）

### 2.4 财务数据（同行业横评时必拉）

指标 | 来源
---|---
营收、归母净利、扣非净利 | 东方财富个股页 F10
毛利率、净利率 | 同上
ROE、资产负债率 | 同上
同比增速 | 同上，至少最近2-3期

### 2.5 机构研报/预测（行业分析时必查）

内容 | 方法
---|---
供需平衡表 | 搜索"XX行业 供需 2026 中信/华鑫/紫金天风"
价格预测 | 搜索"XX价格 展望 2026 伯恩斯坦/高盛"
机构评级 | 搜索"XX股票 评级 目标价"

### 2.6 历史K线（生成走势图时用）

```bash
# 前复权日K线，120条（约半年）
curl -s 'https://web.ifzq.gtimg.cn/appstock/app/fqkline/get?param=sz002460,day,,,120,qfq'
```

## 三、分析层 —— 做什么分析

### 3.1 行业分析维度

```
┌─────────────────────────────────────────┐
│              行业全景分析                 │
├───────────┬───────────┬─────────────────┤
│  宏观层   │  产业层   │    市场层        │
├───────────┼───────────┼─────────────────┤
│ 国际环境  │ 供需平衡  │ 板块行情快照     │
│ 国家政策  │ 库存周期  │ 估值横截面对比   │
│ 资源国博弈│ 产能pipeline│ 机构一致预期   │
│ 替代威胁  │ 成本曲线  │ 价格展望时间轴   │
└───────────┴───────────┴─────────────────┘
```

#### 3.1.1 国际环境
- 主产国出口政策变化（禁运、配额、关税）
- 主要矿山供给扰动（火灾、停产、品位下降）
- 资源民族主义趋势
- 全球需求端地区分布与增速

#### 3.1.2 国家政策
- 矿产资源法规（矿种认定、开发门槛）
- 下游需求刺激政策（新能源车、储能补贴）
- 供给侧管控（环保限产、出口管制）
- 区分：利多 vs 利空，硬性规定 vs 鼓励方向

#### 3.1.3 供需分析
- 搜集3-4家机构供需平衡表，交叉验证
- 判断：供给增速 vs 需求增速 → 过剩/紧平衡/缺口
- 库存趋势（累库/去库）+ 库销比
- 新增产能pipeline与投放节奏

#### 3.1.4 价格展望
- 成本曲线：确定价格底部（如锂15万=云母提锂现金成本线）
- 短期（1-3月）→ 中期（半年）→ 远期（1-3年）三段时间轴
- 机构预测均价（如伯恩斯坦2026锂均价16.7万）

### 3.2 个股分析维度

```
┌──────────────────────────────────────────────┐
│                个股深度分析                     │
├─────────┬─────────┬─────────┬────────────────┤
│ 市场面  │ 财务面  │ 资源面  │   估值面        │
├─────────┼─────────┼─────────┼────────────────┤
│ 现价涨跌│ 营收利润│ 资源禀赋│ PE/PB横评       │
│ 市值换手│ 毛利率  │ 自给率  │ 同业对比         │
│ 周期涨跌│ 负债率  │ 成本优势│ 机构目标价       │
│ K线形态 │ ROE增速 │ 产能规模│ 业绩弹性量化     │
└─────────┴─────────┴─────────┴────────────────┘
```

#### 3.2.1 市场面
- 当日涨跌幅、成交量、换手率 → 判断情绪
- 近1月/3月/6月涨跌幅 → 判断趋势和超跌程度
- PE(TTM)是否合理区间

#### 3.2.2 财务面（至少拉最近一期季报）
- 营收及增速、归母净利及增速
- 毛利率（核心！资源股毛利率=竞争力）
- 资产负债率（财务健康度）
- ROE

#### 3.2.3 资源面（资源股核心差异点）
- 资源自给率（100% vs 50% 天差地别）
- 核心矿山/盐湖名称、品位、成本
- 提锂/冶炼成本（元/吨）
- 产能规模及扩产计划
- 海外政策风险暴露度

#### 3.2.4 估值面
- 同业PE/PB横截面对比（同板块至少拉5-8只）
- 机构目标均价与现价差距
- 业绩弹性量化：商品价格每变动1单位→年化利润增量

## 四、对比与排名层

### 4.1 多维度排名矩阵（行业分析核心产出）

维度 | 说明
---|---
**长期投资价值** | 资源壁垒+成本优势+穿越周期能力
**周期弹性** | 商品涨价时利润扩张速度
**抗周期能力** | 商品跌价时亏损风险
**成本优势** | 提锂/冶炼现金成本排名
**资源安全性** | 自给率+地缘政治风险
**估值修复空间** | 现价 vs 机构目标价

每个维度排名1-2-3，综合给出推荐排序。

### 4.2 分层配置建议

```
长线底仓（40-50%）：资源壁垒最强 + 成本最低的标的
中线弹性（30-40%）：全产业链对冲 + 估值合理的标的
短线博弈（≤15%）：  弹性最大但基本面偏弱的标的
```

## 五、输出层 —— 输出什么结构

### 5.1 行业全景分析输出模板

```
一、价格/行情仪表盘（4卡片：现货价、期货价、库存、排产）
二、国际环境（供给端 + 需求端，利多利空分列）
三、国家政策（利多政策 + 利空因素，逐条标 ✅/❌）
四、供需平衡表（3-4家机构数据对比表）
五、价格展望时间轴（短期→中期→远期）
六、板块暴跌/异动全景表（全部成分股：现价/跌幅/PE/市值/周期涨跌）
七、核心标的PK（3只：财务表 + 资源禀赋表 + 弹性量化 + 优劣总结）
八、名单外潜力股（2-3只，附核心逻辑）
九、综合排名 + 分层配置建议
十、风险提示
```

### 5.2 多股对比输出模板

```
一、三股基本面快照（价格/PE/市值/涨跌 一览表）
二、财务数据对比（营收/利润/毛利率/负债率）
三、资源禀赋对比（自给率/矿山/成本/产能）
四、业绩弹性量化（价格变动→利润增量）
五、多维排名矩阵
六、结论：选谁、什么风格适合什么标的
```

### 5.3 评分信号系统

信号 | 标记 | 含义
---|---|---
强烈推荐 | 🟢 首选 | 基本面+估值+技术面多重共振
关注观察 | 🟡 次选 | 逻辑成立但有瑕疵
谨慎参与 | 🔴 回避 | 基本面有硬伤或纯博弈
防守配置 | 🛡️ | 低估值+高安全边际
弹性博弈 | 🎯 | 小市值+高波动+超跌

## 六、执行原则

1. 所有行情数据从腾讯API实时拉取，不编造、不记忆
2. 政策消息先验证再引用，用 ✅/❌ 标记真伪
3. 同板块至少横评5只以上，不孤立看一只
4. PE必须同业对比，不能跨行业比（银行PE vs 锂矿PE 无意义）
5. 亏损股标"亏损"，不参与PE排名
6. 每次分析必须给明确结论，不模棱两可
7. 结论要说明：什么风格的投资者适合什么标的
