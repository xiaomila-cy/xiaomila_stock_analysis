# 分析维度详解

## 行业分析（三维）

```
┌─────────────────────────────────────────────┐
│               行业全景分析                     │
├─────────────┬─────────────┬─────────────────┤
│   宏观层    │   产业层    │     市场层       │
├─────────────┼─────────────┼─────────────────┤
│ 国际环境    │ 供需平衡    │ 板块行情快照     │
│ 国家政策    │ 库存周期    │ 估值横截面对比   │
│ 资源国博弈  │ 产能pipeline│ 机构一致预期     │
│ 替代威胁    │ 成本曲线    │ 价格展望时间轴   │
└─────────────┴─────────────┴─────────────────┘
```

### 宏观层

- **国际环境**：主产国出口政策（禁运/配额/关税）、矿山供给扰动、资源民族主义
- **国家政策**：矿产资源法规、下游需求刺激（新能源车/储能补贴）、供给管控
- **区分**：利多 vs 利空，硬性规定 vs 鼓励方向

### 产业层

- **供需平衡**：搜集 3-4 家机构数据，交叉验证
- **库存趋势**：累库/去库 + 库销比
- **产能 pipeline**：新增产能与投放节奏

### 市场层

- **板块行情**：全部成分股现价/跌幅/PE/市值
- **估值横评**：同业 PE/PB 分布
- **价格展望**：短期→中期→远期三段时间轴

## 个股分析（四维）

```
┌──────────────────────────────────────────────────┐
│                  个股深度分析                       │
├───────────┬───────────┬───────────┬──────────────┤
│  市场面   │  财务面   │  资源面   │   估值面      │
├───────────┼───────────┼───────────┼──────────────┤
│ 现价涨跌  │ 营收利润  │ 资源禀赋  │ PE/PB横评     │
│ 市值换手  │ 毛利率    │ 自给率    │ 同业对比       │
│ 周期涨跌  │ 负债率    │ 成本优势  │ 机构目标价     │
│ K线形态   │ ROE增速   │ 产能规模  │ 业绩弹性量化   │
└───────────┴───────────┴───────────┴──────────────┘
```

### 市场面
- 当日涨跌幅、成交量、换手率 → 情绪判断
- 近 1/3/6 月涨跌幅 → 趋势和超跌判断
- PE(TTM) 是否合理区间

### 财务面
- 营收及增速、归母净利及增速 → 至少最近一期季报
- **毛利率**（资源股核心！= 竞争力）
- 资产负债率 → 财务健康度
- ROE

### 资源面（资源股差异点）
- 资源自给率：100% vs 50% 天差地别
- 核心矿山/盐湖名称、品位、成本
- 提锂/冶炼现金成本（元/吨）
- 产能规模及扩产计划
- 海外政策风险暴露度

### 估值面
- 同业 PE/PB 横截面对比（至少 5-8 只）
- 机构目标均价 vs 现价差距
- **业绩弹性量化**：商品价格每变动 1 单位 → 年化利润增量
