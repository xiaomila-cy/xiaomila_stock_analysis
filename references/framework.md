### §四估值结果自检 ⬅️ v2.3.2新增

估值表输出后强制执行：

1. **PE偏离度检查**：|标的PE - 同行PE中位数| / 同行PE中位数
   - 偏离度 > 50% → ⚠️ 触发警告，排查：(a)F10标签滞后(b)非经常性损益扭曲(c)确实高估/低估
   - 偏离度 ≤ 50% → ✅ 通过

2. **PB/PS交叉验证**：若PE偏离超50%，用PB/PS做二次验证。PB/PS同样大幅偏离 → 可能选错同行；仅PE偏离 → 可能是利润异常。

### 五.5 极端行情诊断 ⬅️ v2.3新增

**触发条件**：近3个月内的**任一时间段**（可以是任意连续20/30/60个交易日）出现过 ±50%及以上的累计涨跌幅。

> 注意：不是整3个月的涨跌幅，而是滚动窗口——任意连续交易日区间内出现过极端波动即触发。

**诊断内容**：

| 子项 | 内容 |
|------|------|
| 时间窗口定位 | 精确到起止日期，如"2026-03-15 至 2026-04-20，25个交易日" |
| 涨幅/跌幅 | 精确百分比 |
| 驱动因素判定 | 按以下框架逐条排查，标注✅/❌： |

**上涨≥50% 诊断清单：**

| 可能原因 | 判定方法 | 结果 |
|----------|----------|------|
| 业绩大幅超预期 | 检查该时段前后是否有业绩预告/快报/正式财报，增速是否显著超一致预期 | ✅/❌ |
| 踩中市场主线热点 | 检查该时段市场主线热点（如AI/新能源/周期等），标的所属概念是否与主线重合 | ✅/❌ |
| 重大订单/合作公告 | 检查该时段是否有重大合同/战略合作/新产品发布公告 | ✅/❌ |
| 重组/借壳/资产注入预期 | 检查是否有重组进展公告或市场传闻（注来源） | ✅/❌ |
| 行业政策大利好 | 检查是否有直接利好该行业的政策发布 | ✅/❌ |
| 资金驱动/情绪驱动 | 排除以上因素后，成交量+换手率是否异常放大（纯资金博弈） | ✅/❌ |

**下跌≥50% 诊断清单：**

| 可能原因 | 判定方法 | 结果 |
|----------|----------|------|
| 业绩暴雷 | 检查该时段前后是否发布业绩大幅低于预期/预亏公告 | ✅/❌ |
| 行业政策利空 | 检查是否有直接打击该行业的政策（集采/监管/关税等） | ✅/❌ |
| 重大诉讼/处罚/违规 | 检查是否有立案调查/行政处罚/重大诉讼公告 | ✅/❌ |
| 股东大规模减持 | 检查是否有大股东/高管减持公告，减持比例是否显著 | ✅/❌ |
| 行业周期拐点 | 检查行业供需是否出现逆转信号（价格暴跌/库存积压） | ✅/❌ |
| 流动性危机 | 检查是否伴随成交量骤降/融资爆仓/股权质押预警 | ✅/❌ |
| ST/退市风险 | 检查是否触发ST警示/退市风险警示 | ✅/❌ |

**诊断结论**：
- 用一句话定性：这是"业绩驱动"还是"热点驱动"还是"资金驱动"还是"黑天鹅"
- 判断该涨跌幅是否可持续/可逆转
- 标注"该事件是否已在当前股价中充分定价"

### 九.3 配置建议（条件触发式 ⬅️ v2.5.0）

> 本节的核心理念：**不是"给一个价格区间"，而是"给一套你可以照着执行的条件触发规则"**。
> 学习自DSA Dashboard的battle_plan设计——当条件X满足时执行动作Y。

#### 最终评分与定性

| 子项 | 格式 |
|------|------|
| 最终评分 | 基础分__ × 系数__ = __/50 |
| 一句话定性 | 定位 + 投资逻辑，不超过30字 |
| 风格匹配 | 🟢长线价值 / 🟡趋势波段 / 🔴短线博弈 / ⛔不建议 |

#### 仓位执行规则（条件触发式）

```
IF  [触发条件A] → THEN [执行动作A]
IF  [触发条件B] → THEN [执行动作B]
...
ELSE → 维持当前仓位或观望
```

**每条规则必须包含三个元素**：
1. **触发条件**：可量化、可观测（价格突破XX元 / 财报增速<XX% / 板块指数跌破XX）
2. **执行动作**：具体的仓位调整（加仓X% / 减仓X% / 清仓 / 止损）
3. **理由**（可选但推荐）：一句话说明为什么这个条件重要

**模板**：

```
📋 建仓规则：
  触发：{价格/事件条件} → 动作：建仓X%，理由：{一句话}
  触发：{价格/事件条件} → 动作：加仓至X%，理由：{一句话}

🛑 减仓/止损规则：
  触发：{价格/事件条件} → 动作：减仓至X% / 清仓，理由：{一句话}
  触发：{价格/事件条件} → 动作：无条件止损，理由：{一句话}

📅 关注节点：
  · {日期/事件}：{关注什么，以及触发后该怎么做}
  · {日期/事件}：{关注什么}
```

**输出示例**（贵州茅台）：

```
📋 建仓规则：
  触发：价格站稳MA20(¥1,212)上方3个交易日 → 建仓5%，理由：中期趋势修复信号
  触发：Q2营收增速>8% + 价格>¥1,250 → 加仓至15%，理由：基本面+技术面双重确认
  触发：价格回踩¥1,150不破 + 北向连续3日净流入 → 加仓5%，理由：极限估值支撑+聪明钱信号

🛑 减仓/止损规则：
  触发：价格跌破¥1,150 → 减半仓，理由：前低支撑失效
  触发：价格跌破¥1,100 → 清仓，理由：2026年低点破位，趋势逆转
  触发：白酒板块指数单周跌幅>3% → 暂停加仓，理由：板块系统性风险
  触发：单日跌幅>5%且放量 → 次日开盘减半仓，理由：恐慌性抛售信号

📅 关注节点：
  · 8月 Q2财报：营收增速<8% → 下调目标仓位至10%；增速>12% → 上调至20%
  · 分红派息日：除权后观察填权速度，5日内未填权→暂时观望
  · 消费税改革进展：实质性利空政策出台→重新评估整个白酒板块
```

#### 风控清单（从配置规则中抽取，集中呈现）

最终报告H5中，将上述规则的风控部分抽取为5条以内的精简清单，放在执行摘要中。格式：

```
⚠️ 核心风控（5条）：
1. ¥1,150 = 前低止损线，跌破无条件执行
2. 单日跌幅>5% = 恐慌信号，次日减半仓
3. 板块周跌>3% = 系统性风险，暂停加仓
4. Q2营收<8% = 基本面不及预期，降仓位上限
5. 上证破3100 = 大盘风险，茅台同步减仓
```

## 十、三视角交叉验证

### 执行原则

- **时机**：完成§一至§八全部数据采集和分析后，§九综合结论之前
- **原则**：同一份真实数据，三个不同角色立场各自独立分析。不换模型，换立场
- **强制差异化要求**（v2.3强化）：
  - 🔵 多头须至少列出1条"市场定价错误的证据"
  - 🔴 空头须至少列出1条"最坏情况下的极限亏损测算"
  - ⚪ 审计须至少指出1处"数据矛盾或逻辑跳跃"
  - **不达标 = 未通过，需重做。** 三次重做仍不达标 → 置信度强制🔴
- **v2.4.0 新增验证项**：⚪ 审计必须核对 §二管理层展望 vs 实际动作（管理层说要做的事 vs 公告/季报显示的实际进展），不一致处标⚠️

### 三视角定义

| 视角 | 角色设定 | 默认立场 | 核心问题 |
|------|----------|----------|----------|
| 🔵 多头 | 价值投资者 | 这只票被市场低估了 | "如果这是一只好股票，证据在哪？市场错在哪？" |
| 🔴 空头 | 趋势交易者/怀疑者 | 市场定价大概率正确 | "如果你错了，哪里会出错？最坏情况是什么？" |
| ⚪ 审计 | 独立审计师 | 不判断多空，只看逻辑一致性 | "数据之间有没有矛盾？推理有没有跳跃？假设合不合理？" |

### 输出格式

```
🟢 高置信度结论（三视角一致）：
  - ...（具体判断，引用数据）
  - ...
🟡 中等置信度结论（两票对一票）：
  - ...（多头+审计同意，空头反对，理由是...）
🔴 低置信度/盲区（三视角分歧）：
  - ...（三视角各自判断不同，建议降低该维度对决策的权重）
```

## 十一、行业权重调节表 ⬅️ v2.3新增

不同行业基本面特征差异大，评分权重不能一刀切。以下为行业权重参考表：

| 行业类型 | 基本面对 | 财务 | 估值 | 政策 | 管理 | ESG |
|----------|---------|------|------|------|------|-----|
| 周期/资源（有色/煤炭/化工/钢铁） | 7 | 8 | 12 | 10 | 5 | 8 |
| 成长/科技（半导体/AI/新能源/创新药） | 12 | 7 | 8 | 10 | 8 | 5 |
| 消费/医药（食品饮料/家电/中药） | 10 | 10 | 10 | 7 | 8 | 5 |
| 金融/地产（银行/保险/券商/地产） | 8 | 10 | 12 | 12 | 5 | 3 |
| ST/困境反转 | 8 | 5 | 8 | 15 | 6 | 8 |

**使用说明**：
- 取行业类型对应的权重，归一化到各维度满分（基本面10/财务10/估值10/政策10/管理层5/ESG5）
- 周期股估值权重高（PB比PE重要），成长股基本面权重高（赛道空间>当前利润）
- ST股政策权重最高（摘帽/重整是核心变量）
- 若标的跨多个类型，取最贴合的一个

## 十二、特殊标的适配层 ⬅️ v2.3新增

### ST/*ST 股
- 基础分析用PB/PS替代PE（PE失效）
- 新增：**退市风险清单**（连续亏损年限/净资产是否负数/审计意见类型/立案调查状态）
- 新增：**重整/重组进展时间线**（申请日/受理日/方案公告日/债权人会议日/法院裁定日）
- 新增：**摘帽条件核查表**（盈利+净资产>1元+审计标准无保留+无立案调查→逐条打✅/❌）
- §七政策权重提升至15分（ST的核心是政策和监管博弈）

### 亏损股
- PE失效，估值侧重PB/PS/重置成本法
- 财务面重点：亏损收窄趋势、现金流是否断裂、融资能力

### 次新股（上市不足1年）
- 缺乏历史财务数据 → 侧重行业空间+管理团队背景
- 缺乏长期K线 → 用上市后全部K线+同行业老股对标
- 重点：解禁日/解禁量（3个月/6个月/1年三个节点）

### 周期股
- 财务面侧重毛利率趋势（不是绝对值）
- 估值面：PE在周期顶部失效，用PB+商品价格位置判断
- §五.5极端行情诊断 = 强相关（周期股大涨大跌是常态，需区分"周期波动"vs"趋势逆转"）

### 双主业/跨行业转型股 ⬅️ v2.3.2新增
- **触发条件**：年报业务构成中有两条及以上业务线各自营收占比≥20%，且行业属性差异大（如家纺+化工、制造+科技）
- **对标方式**：对每条主要业务线分别选取同行进行估值对标，最终取业务营收加权平均作为估值参考
- **行业权重**：取两条业务线对应行业的权重均值（四舍五入取整）。例：家纺（消费10/10/10/7/8/5）+ 化工（周期7/8/12/10/5/8）→ 均值（9/9/11/9/7/7）
- **F10陷阱提醒** ⬅️ v2.3.2：年报业务构成才是法律文件，东财F10标签可能滞后。若F10标签与年报明显不符，标注"⚠️F10标签滞后"，以年报为准
- **特别关注**：新业务板块若营收占比小（<10%）但利润贡献大（>30%），应标注为"弹性引擎"——估值时应给予新业务更高的权重
