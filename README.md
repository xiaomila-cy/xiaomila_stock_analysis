## 这是什么

金叉是一套完整的 A 股分析工作流，让 AI Agent 能像专业分析师一样干活：

📊 数据采集
腾讯行情 API · 东方财富数据 · 机构研报 · 商品现货价格

🔬 分析框架
宏观→产业→市场三层 + 个股四维（市场/财务/资源/估值）

📋 输出模板
行业全景报告 · 多股对比 · 评分信号系统 · 分层配置建议

---

## 分析体系总览

```
一、触发场景 → 四种分析模式自动匹配
二、数据采集 → 6 类数据源（行情/现货/政策/财务/研报/K线）
三、分析层   → 行业三维 + 个股四维
四、排名层   → 6 维度矩阵 + 分层配置
五、输出层   → 行业全景/多股对比 两大模板
```

## 板块详情

### 一、触发场景

| 用户输入 | 触发模式 |
|---------|---------|
| "分析下 XX 股票" | 个股深度分析 |
| "XX 行业怎么看" | 行业全景分析 |
| "这三只哪个好" | 多股对比分析 |
| "最近 XX 跌了好多" | 事件驱动分析 |

### 二、数据采集层

- **行情数据**：腾讯 `qt.gtimg.cn` API，实时价格 + 周期涨跌幅（GP-A 定位法）
- **商品现货**：东方财富搜索 → Mysteel/生意社报价
- **政策验证**：拆分陈述 → 原文对照 → ✅/❌ 真伪标记
- **财务数据**：东方财富 F10（营收/毛利率/ROE/负债率）
- **机构研报**：供需平衡表 + 价格预测 + 目标评级
- **历史 K 线**：前复权日线，120 条（约半年）

### 三、分析层

**行业全景（三层）：**

```
宏观层 → 国际环境 · 国家政策 · 资源国博弈
产业层 → 供需平衡 · 库存周期 · 产能 pipeline · 成本曲线
市场层 → 板块行情快照 · 估值横截面 · 机构预期 · 价格展望
```

**个股深度（四维）：**

```
市场面 → 价格 · 成交量 · 换手率 · 周期涨跌 · PE 估值
财务面 → 营收增速 · 毛利率（核心）· 负债率 · ROE
资源面 → 自给率 · 矿山品位 · 提锂成本 · 产能扩产
估值面 → 同业 PE/PB 横评 · 机构目标价 · 业绩弹性量化
```

### 四、对比与排名

**6 维度排名矩阵：**

长期投资价值
资源壁垒 + 成本优势 + 穿越周期

周期弹性
商品涨价时利润扩张速度

抗周期能力
商品跌价时亏损风险

成本优势
提锂/冶炼现金成本排名

资源安全性
自给率 + 地缘政治风险

估值修复空间
现价 vs 机构目标价

**分层配置建议：**

长线底仓(40-50%)
资源壁垒最强 + 成本最低

中线弹性(30-40%)
全产业链对冲 + 估值合理

短线博弈(≤15%)
弹性最大但基本面偏弱

### 五、输出模板

**行业全景分析（10 模块）：**
价格仪表盘 → 国际环境 → 国家政策 → 供需平衡表 → 价格展望 → 板块全景表 → 核心标的 PK → 潜力股 → 综合排名 → 风险提示

**评分信号系统：**

🟢 首选
基本面+估值+技术面多重共振

🟡 次选
逻辑成立但有瑕疵

🔴 回避
基本面有硬伤或纯博弈

🛡️ 防守
低估值+高安全边际

🎯 弹性
小市值+高波动+超跌

## 执行原则

1. 所有行情数据从腾讯 API 实时拉取，不编造、不记忆
2. 政策消息先验证再引用，用 ✅/❌ 标记真伪
3. 同板块至少横评 5 只以上，不孤立看一只
4. PE 必须同业对比，不跨行业比
5. 每次分析必须给明确结论，不模棱两可
